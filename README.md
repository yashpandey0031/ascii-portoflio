# ASCII Portfolio Landing

Retro, terminal-inspired landing screen built with:

- Next.js (App Router)
- Tailwind CSS (strict grayscale)
- GSAP (boot progression + sequencing)
- Framer Motion (minimal reveal transitions)

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Current scope

Implemented only the landing experience for now:

- terminal boot logs
- ASCII loading/progress indicator
- typewriter prompt intro
- custom ASCII cursor with blink + hover state
- monochrome-only visual system

Future sections (about, experience, projects, education, hobbies) can be added next.
